Operator Categories

Transformation

  • map - Transform values
  • switchMap - Switch to new Observable
  • mergeMap - Merge multiple Observables
  • concatMap - Concatenate Observables
  • scan - Accumulate values

Filtering

  • filter - Filter values by condition
  • take - Take first n values
  • distinctUntilChanged - Remove duplicates
  • debounceTime - Debounce emissions

Utility

  • tap - Side effects
  • startWith - Start with initial values
  • catchError - Error handling

Combining

  • forkJoin - Wait for all to complete
  • combineLatest - Latest from each source
  • merge - Merge all emissions
  • concat - Sequential concatenation
  • zip - Pair by index
  • race - First to emit wins
